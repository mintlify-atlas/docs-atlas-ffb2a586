---
title: Label
description: A versatile label component for tags, badges, and dismissible chips with multiple variants and appearances.
---

## Import

```vue
import { MtLabel } from '@/components/_internal/mt-label.vue';
```

## Props

<ParamField path="variant" type="string">
  The semantic variant determining the label's color scheme.
  
  **Options:** `info`, `danger`, `success`, `warning`, `neutral`, `primary`
</ParamField>

<ParamField path="size" type="string" default="default">
  The size of the label.
  
  **Options:** `small`, `medium`, `default`
  
  **Sizes:**
  - `small`: 16px height
  - `medium`: 24px height  
  - `default`: 32px height
</ParamField>

<ParamField path="appearance" type="string" default="default">
  The visual style of the label.
  
  **Options:** `default`, `pill`, `circle`, `badged`
  
  - `default`: Standard rectangular label with rounded corners
  - `pill`: Fully rounded ends
  - `circle`: Circular label (24px × 24px)
  - `badged`: Transparent label with color badge indicator
</ParamField>

<ParamField path="ghost" type="boolean" default="false">
  When true, removes the background color for a ghost/outline appearance.
</ParamField>

<ParamField path="caps" type="boolean" default="false">
  When true, transforms text to uppercase.
</ParamField>

<ParamField path="dismissable" type="boolean" default="false">
  When true and dismiss event handler is present, shows a close button on hover.
</ParamField>

## Events

<ParamField path="dismiss" type="event">
  Emitted when the dismiss button is clicked (only when `dismissable` is true).
</ParamField>

<ParamField path="selected" type="event">
  Emitted when the label is clicked.
</ParamField>

## Slots

<ParamField path="default" type="slot">
  The main content area for label text.
</ParamField>

<ParamField path="dismiss-icon" type="slot">
  Custom icon for the dismiss button (replaces default close icon).
</ParamField>

## Usage Examples

### Basic Labels

```vue
<template>
  <mt-label>Default</mt-label>
  <mt-label variant="info">Info</mt-label>
  <mt-label variant="success">Success</mt-label>
  <mt-label variant="warning">Warning</mt-label>
  <mt-label variant="danger">Danger</mt-label>
</template>
```

### Different Sizes

```vue
<template>
  <mt-label size="small">Small</mt-label>
  <mt-label size="medium">Medium</mt-label>
  <mt-label size="default">Default</mt-label>
</template>
```

### Appearance Variants

```vue
<template>
  <mt-label appearance="default" variant="info">
    Default
  </mt-label>
  
  <mt-label appearance="pill" variant="success">

  </mt-label>
  
  <mt-label appearance="circle" variant="warning">

  </mt-label>
```

### Ghost Labels

```vue

    Ghost Info
  </mt-label>
  <mt-label variant="success" :ghost="true">
    Ghost Success
  </mt-label>
</template>
```

### Dismissible Labels (Tags)

```vue
<template>
  <mt-label 
    v-for="tag in tags"
    :key="tag.id"
    variant="info"
    :dismissable="true"
    @dismiss="removeTag(tag.id)"
  >
    {{ tag.name }}
  </mt-label>
</template>

<script setup>
import { ref } from 'vue';

const tags = ref([
  { id: 1, name: 'JavaScript' },
  { id: 2, name: 'Vue.js' },
  { id: 3, name: 'TypeScript' },
]);

const removeTag = (id) => {
  tags.value = tags.value.filter(tag => tag.id !== id);
};
</script>
```

### Clickable Labels

```vue
<template>
  <mt-label 
    variant="primary"
    @selected="handleLabelClick"
  >
    Click me
  </mt-label>
</template>

<script setup>
const handleLabelClick = () => {
  console.log('Label clicked');
};
</script>
```

### Uppercase Labels

```vue
<template>
  <mt-label :caps="true" variant="info">
    uppercase text
  </mt-label>
</template>
```

### Custom Dismiss Icon

```vue
<template>
  <mt-label 
    variant="danger"
    :dismissable="true"
    @dismiss="handleDismiss"
  >
    Custom Icon
    <template #dismiss-icon>

    </template>
  </mt-label>
</template>
```

### Label Group

```vue
<template>
  <div style="display: flex; flex-wrap: wrap; gap: 8px;">
    <mt-label 
      v-for="category in categories"
      :key="category.id"

      appearance="pill"

      {{ category.name }}
    </mt-label>

</template>

<script setup>
const categories = [
  { id: 1, name: 'Frontend', variant: 'info' },
  { id: 2, name: 'Backend', variant: 'success' },
  { id: 3, name: 'DevOps', variant: 'warning' },
  { id: 4, name: 'Design', variant: 'primary' },
];
</script>
```

## Behavior Notes

- Labels have `tabindex="0"` making them keyboard accessible
- Dismiss button only appears on hover when `dismissable` is true
- Text automatically truncates with ellipsis when content exceeds width
- Minimum width of 56px ensures consistent sizing
- Default margin: `0 6px 6px 0` for natural spacing in groups

## Styling

- Hover effect on dismissible labels: changes to brand colors
- Smooth transitions for hover states
- Ghost variant: transparent background with colored border
- Badged appearance: includes color-coded badge indicator
