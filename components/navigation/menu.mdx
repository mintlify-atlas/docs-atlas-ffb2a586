## Overview

The Meteor component library provides two menu systems: Action Menu (mt-action-menu) for dropdown menus with rich features like keyboard shortcuts and external links, and Context Menu (mt-context-button) for simpler popover-based menus. Both support menu items, grouping, and various states.

## Action Menu

### Import

```vue
import { DropdownMenu, DropdownMenuTrigger } from 'reka-ui';
import MtActionMenu from '@/components/action-menu/mt-action-menu/mt-action-menu.vue';
import MtActionMenuItem from '@/components/action-menu/mt-action-menu-item/mt-action-menu-item.vue';
import MtActionMenuGroup from '@/components/action-menu/mt-action-menu-group/mt-action-menu-group.vue';
```

### Action Menu Props

<ParamField path="isSubMenu" type="boolean" default="false">
  Renders the menu as a submenu with adjusted positioning
</ParamField>

<ParamField path="matchTriggerWidth" type="boolean" default="false">
  Makes the menu content match the trigger button's width
</ParamField>

### Action Menu Item Props

<ParamField path="variant" type="'default' | 'critical'" default="'default'">
  Visual style variant. Use 'critical' for destructive actions
</ParamField>

<ParamField path="icon" type="string">
  Icon name to display before the item label
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the menu item
</ParamField>

<ParamField path="shortcut" type="ShortcutDefinition">
  Keyboard shortcut to display and announce for accessibility
</ParamField>

<ParamField path="isSubTrigger" type="boolean" default="false">
  Renders the item as a submenu trigger with an arrow indicator
</ParamField>

<ParamField path="link" type="string">
  External URL. Renders the item as a link that opens in a new tab
</ParamField>

### Shortcut Definition

```typescript
interface ShortcutDefinition {
  modifiers?: ('mod' | 'ctrl' | 'alt' | 'shift' | 'meta')[];
  key: string | 'enter' | 'esc' | 'tab' | 'space' | 'backspace' | 'delete' | 'up' | 'down' | 'left' | 'right';
}
```

## Context Menu

### Import

```vue
import MtContextButton from '@/components/context-menu/mt-context-button/mt-context-button.vue';
import MtContextMenuItem from '@/components/context-menu/mt-context-menu-item/mt-context-menu-item.vue';
```

### Context Button Props

<ParamField path="icon" type="string" default="'solid-ellipsis-h-s'">
  Icon name for the menu trigger button
</ParamField>

<ParamField path="title" type="string" default="''">
  Title displayed at the top of the menu popover
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the menu button
</ParamField>

<ParamField path="hasError" type="boolean" default="false">
  Shows an error state on the button
</ParamField>

### Context Menu Item Props

<ParamField path="label" type="string" required>
  Text label for the menu item
</ParamField>

<ParamField path="icon" type="string">
  Icon name to display before the label
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the menu item
</ParamField>

<ParamField path="type" type="'default' | 'active' | 'critical'" default="'default'">
  Visual style variant
</ParamField>

### Dynamic Menu Items

```vue
<template>
  <mt-context-button>
    <mt-context-menu-item
      v-for="action in availableActions"
      :key="action.id"
      :label="action.label"
      :icon="action.icon"
      :type="action.type"
      :disabled="action.disabled"
      @click="handleAction(action)"
    />
  </mt-context-button>
</template>

<script setup>
import { computed } from 'vue';

const availableActions = computed(() => [
  { id: 'view', label: 'View Details', icon: 'eye-s', type: 'default' },
  { id: 'edit', label: 'Edit', icon: 'edit-s', type: 'default' },
  { id: 'archive', label: 'Archive', icon: 'archive-s', type: 'default' },
  { id: 'delete', label: 'Delete', icon: 'trash-s', type: 'critical' }
]);

const handleAction = (action) => {
  console.log('Action clicked:', action.id);
};
</script>
```

## Action Menu Features

- Keyboard navigation support
- Keyboard shortcut display with platform-specific symbols (⌘ on Mac, Ctrl on Windows)
- External link support with automatic target="_blank"
- Submenu support with chevron indicators
- Automatic icon alignment in groups
- ARIA attributes for accessibility

## Context Menu Features

- Simpler popover-based implementation
- Integrates seamlessly with Card component
- Customizable trigger button
- Auto-close on item click
