# Icon Kit Overview

The Meteor Icon Kit is a comprehensive icon library designed specifically for Shopware's product ecosystem. It provides a minimal yet highly expressive visual style that seamlessly integrates with the Meteor Design System.

## What is the Icon Kit?

The Icon Kit is a collection of professionally designed SVG icons that can be easily integrated into your web applications. Each icon is crafted to maintain visual consistency and clarity across different sizes and contexts.

## Key Features

- **Two Style Variants**: Choose between regular (outlined) and solid (filled) styles
- **Extensive Library**: Over 880+ icons covering common use cases
- **Optimized SVGs**: Clean, lightweight SVG files for optimal performance
- **Dynamic Colors**: Icons support dynamic color changes via CSS
- **Framework Agnostic**: Use with any JavaScript framework or plain HTML

## Icon Styles

The Icon Kit provides two distinct visual styles:

### Regular (Outlined)

The regular style features outlined icons with a clean, minimal aesthetic. Perfect for interfaces that need a lighter visual weight.

- **Total icons**: 461
- **Use case**: Primary navigation, toolbars, subtle UI elements
- **Visual weight**: Light

### Solid (Filled)

The solid style offers filled icons with more visual prominence. Ideal for drawing attention and creating emphasis.

- **Total icons**: 425
- **Use case**: Active states, primary actions, emphasis
- **Visual weight**: Bold

## Icon Categories

The Icon Kit includes icons across multiple categories:

- **Navigation**: arrows, chevrons, bars
- **Actions**: edit, delete, copy, save
- **Communication**: mail, message, bell, phone
- **Commerce**: shopping-cart, wallet, credit-card, tag
- **Media**: image, video, camera, file
- **Interface**: settings, search, filter, more
- **Social**: share, like, bookmark, star
- **Status**: check, warning, error, info
- **And many more**: 3D, AR, analytics, charts, calendar, and specialized icons
